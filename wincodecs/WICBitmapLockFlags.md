title: WICBitmapLockFlags

Specifies access to an IWICBitmap.

```cpp
typedef enum WICBitmapLockFlags {
  WICBitmapLockRead,
  WICBitmapLockWrite,
  WICBITMAPLOCKFLAGS_FORCE_DWORD
} ;
```

## WICBitmapLockRead

A read access lock.

## WICBitmapLockWrite

A write access lock.
